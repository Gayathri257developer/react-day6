4. How reducer as pure function in redux ?

In redux, a reducer is a pure function. these function accepts 2 arguments state and action. it always return a new state based on action, does not change the existing state.

5. Why we use redux thunk?

It allows us to dispatch actions manually, which gives us the power to incorporate some logic or run some asynchronous code before dispatching an action.
